# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [0.1.0] - 2026-01-13
### Added
- Floating dashboard for browsing installed colorschemes.
- Live preview with instant apply and safe revert on cancel.
- Lazy.nvim installer command to add theme repos.
- Minimal sizing config and optional toggle keymap.
